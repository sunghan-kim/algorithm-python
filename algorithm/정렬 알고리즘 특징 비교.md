# 정렬 알고리즘 특징 비교

| 정렬 알고리즘           | 특징                                                         | 시간 복잡도                                                  |
| ----------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 버블 정렬 (Bubble Sort) | - 인접한 데이터 비교<br />- 앞의 데이터가 뒤의 데이터보다 크면 자리 바꿈 | - 평균 : $O(n^2)$<br />- 최악 : $\frac{n (n-1)}{2}$<br />- 최선 : $O(n)$ |
